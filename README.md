# 🧠 TAT-ROOT

## Что это

TAT-ROOT — это публичная часть экосистемы TAT.
Содержит только безопасные для публикации модули — обёртки, демонстрации и документацию.

## Структура

```
TAT-ROOT/
├── tat_sabbath/          # Модуль верификации (обёртка, безопасно)
├── README.md             # Описание
└── LICENSE               # AGPL-3.0 + коммерческая лицензия
```

## Ядро TAT

Ядро системы (TAT_ROOT, обученные модели, состояния) не публикуется.
Для коммерческого использования и доступа к полной версии — свяжитесь с автором.
