# TAT-ROOT

**TAT — Biological Memory for LLMs**

TAT stores memory as a living system:
- **Mitosis** — chunks divide at 50 KB
- **Apoptosis** — obsolete chunks archive
- **Coherence** — θ=1.987
- **Holographic recovery** — from 42% of data

Built without grants, without a team, without a laptop — from a phone and free Colab.
