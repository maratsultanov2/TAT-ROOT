# Архитектура TAT

TAT (TreeAngleTap) — это когнитивно-семантическая архитектура памяти для LLM.

## Принципы

1. **Связи важнее признаков** — переходы важнее состояний.
2. **Голографическая память** — 42% данных достаточно для 100% смысла.
3. **Адаптивная когеренция** — пороги 0.3/0.5/0.7/0.9.
4. **Суббота (SABBATH)** — право на тишину при C < 0.3.
5. **Карусель чанков** — перераспределение веса при насыщении.

## Компоненты

- **Апостолы** — 12 специализированных модулей.
- **Логос** — финальное кольцо.
- **Сатана** — брандмауэр.
- **Кольца** — итеративная обработка.
- **Чанки** — многомерные объекты с 5 плоскостями.

## Математика

- θ = 1.987 (температура когеренции)
- 37/73 (мягкие границы)
- C = |⟨e1 + i·e1·sin(θ), e2 + i·e2·sin(θ)⟩| / (||e1'||·||e2'|| + ε)
